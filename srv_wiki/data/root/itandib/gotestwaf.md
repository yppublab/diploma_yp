# GoTestWAF
## Тестирование
После изменений конфигурации WAF обязательно тестируем эффективность новых настроек / решений. Объективный контроль помогает понять насколько гипотетические улучшения сказались на реальной эффективности WAF.

## GoTestWAF
Существует разнообразие решений по тестированию WAF. Мы в компании придерживаемся решения [GoTestWAF](https://github.com/wallarm/gotestwaf). Конечно, вы можете провести тестирование и другими средствами, однако, для прозрачности результатов оптимально использовать одно решение, не пренибригая другими.

## Подготовка к запуску
Для минимизации трудозатрат на установку и настройку мы всегда действуем оптимальным путем:
```
cd /opt/
docker pull wallarm/gotestwaf
```

## Запуск
```
docker run --rm --network="15_dmz_net" -v ${PWD}/reports:/app/reports wallarm/gotestwaf --url=http://192.168.4.11/ --noEmailReport
```

## Результат
Программа будет тестировать WAF какое-то непродолжительное время. В результате вы получите отчет в форматах PDF и CSV, которые будут лежать в папке /opt/reports. 
Для понимания эффективности идеально сравнивать результаты до и после, или с одним решением, а затем с другим.
